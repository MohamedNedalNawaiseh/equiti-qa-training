Prerequisites

What you need before starting this training program.

Computer Requirements
  • Operating System: Windows 10/11, macOS 10.14+, or Ubuntu 18.04+
  • RAM: Minimum 8GB (16GB recommended for heavy automation)
  • Storage: At least 10GB free space for Node.js, browsers, and projects
  • Processor: Any modern multi-core processor (Intel i5 / Ryzen 5 or better)
  • Internet: Stable internet connection for downloading tools and dependencies
Knowledge Requirements
No Prior Experience Required!
This training is designed for complete beginners. We'll teach you everything from the ground up. However, basic computer literacy and willingness to learn are essential.

Helpful but Not Required:

  • Basic understanding of what software testing is
  • Familiarity with any programming language (even basic HTML/CSS)
  • Understanding of how websites and applications work
  • Any experience with command line/terminal
